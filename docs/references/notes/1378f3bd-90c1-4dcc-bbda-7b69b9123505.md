# Implement GitHub API client module

Create a module that authenticates with the GitHub API and fetches issues.

## Scope
- `src/github.rs` module
- Reads `GITHUB_TOKEN` from env (error if missing)
- Functions: `list_issues(owner, repo, state, page)` and `get_issue(owner, repo, number)`
- Return types using octocrab's built-in models
- Support filtering by state (open/closed/all)
